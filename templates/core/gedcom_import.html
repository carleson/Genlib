Vad är en GEDCOM-fil?

GEDCOM (Genealogical Data Communication) är ett standardformat för att utbyta släktforskningsdata mellan olika program. De flesta släktforskningsprogram kan exportera data till GEDCOM-format.

Vanliga problem
  • Om importen misslyckas, kontrollera att filen är i rätt format (.ged eller .gedcom)
  • Personer utan namn hoppas över automatiskt
  • Relationer som refererar till personer som inte finns ignoreras
  • Detaljerade felmeddelanden visas efter importen och loggas i systemet
